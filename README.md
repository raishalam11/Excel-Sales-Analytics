# 📊 Sales & Finance Analytics Project (Excel)

## Summary


# 📌 Project Overview

This project demonstrates how data analytics can transform raw sales and financial data into actionable business insights.

Using Microsoft Excel’s advanced analytics tools, I developed a comprehensive reporting solution for AtliQ Hardware, enabling stakeholders to monitor sales performance, market trends, and financial outcomes.

The project applies data cleaning, modeling, analysis, and reporting techniques to build interactive business reports that support data-driven decision-making.

The analysis answers important business questions such as:
- Which customers contribute the most revenue?
- How does market performance compare to sales targets?
- What factors impact profitability across markets and time periods?
- How can businesses optimize their sales and financial strategies?

# 🛠 Tools & Technologies

| Tool                | Purpose                                             |
| ------------------- | --------------------------------------------------- |
| **Microsoft Excel** | Data analysis and reporting                         |
| **Power Query**     | Data extraction, transformation, and cleaning (ETL) |
| **Power Pivot**     | Data modeling and relationship building             |
| **Pivot Tables**    | Analytical reporting and dashboard creation         |
| **Data Modeling**   | Creating relationships between datasets             |

# 🎯 Project Objectives

**The key objectives of this project were:**

- Clean and transform raw sales datasets using Power Query
- Build a structured data model using Power Pivot
- Analyze customer and market performance
- Track sales performance against targets
- Create Profit & Loss reports for financial insights
- Develop clear and intuitive reports for business stakeholders
